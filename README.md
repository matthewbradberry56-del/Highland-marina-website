 # Highland Marina Website

Official website repository for **Highland Marina** in LaGrange, Georgia.

## Overview
This project contains the source code and assets for the Highland Marina website.  
It is used for development, testing, and deployment of public-facing marina content.

## Features (Planned / In Progress)
- Marina information & amenities
- Slip rentals & availability
- Events & announcements
- Contact information
- Mobile-friendly design
